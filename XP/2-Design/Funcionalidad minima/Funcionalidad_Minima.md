# Funcionalidad Mínima – AntCiberDron (XP)

La funcionalidad mínima define el conjunto indispensable de capacidades que el sistema debe poseer para cumplir su objetivo principal desde la primera iteración.

## Funcionalidades Esenciales

| ID | Funcionalidad | Descripción |
|----|--------------|-------------|
| FM-01 | Autenticación | Acceso con usuario y contraseña (máx. 3 intentos) |
| FM-02 | Identificación del equipo | Presentación de cédulas y nombres del grupo |
| FM-03 | Lectura de CSV | Carga del archivo `Grupo##.csv` |
| FM-04 | Visualización de loading | Progreso de lectura por coordenada |
| FM-05 | Procesamiento de coordenadas | Lectura línea por línea del archivo |
| FM-06 | Autómata BBA | Evaluación del lenguaje del tipo de arsenal |
| FM-07 | Decisión de explosión | Retorno True o False |
| FM-08 | Ubicación de explosión | Presentación de la coordenada afectada |
| FM-09 | Hormiguero virtual | Creación básica de hormigas |

Estas funcionalidades aseguran que el sistema **cumpla el enunciado completo** y entregue valor inmediato, respetando los principios de XP.
