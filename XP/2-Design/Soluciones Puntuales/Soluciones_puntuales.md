# Soluciones Puntuales – AntCiberDron (XP)

Las soluciones puntuales permiten resolver problemas específicos del sistema de forma directa, evitando sobreingeniería.

## Problemas y Soluciones Aplicadas

| Problema Identificado | Solución Puntual |
|----------------------|------------------|
| Control de acceso | Autenticación simple con límite de intentos |
| Gestión de datos | Uso de archivo CSV como fuente de información |
| Validación de arsenal | Autómata Finito Determinista |
| Complejidad del sistema | Separación por módulos funcionales |
| Decisión de ataque | Implementación de la bomba BBA |
| Seguimiento del proceso | Loading visual durante la lectura |
| Escalabilidad | Uso de herencia para tipos de hormigas |
| Trazabilidad | Registro de coordenadas evaluadas |

Estas soluciones se alinean con XP al ser **claras, específicas y fáciles de modificar** en iteraciones futuras.
