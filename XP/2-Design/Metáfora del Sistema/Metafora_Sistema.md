# Metáfora del Sistema – AntCiberDron (XP)

La metáfora en XP permite que todos los involucrados comprendan el sistema de manera uniforme, usando una analogía compartida.

## Metáfora General
**El sistema AntCiberDron funciona como un hormiguero de guerra cibernético inteligente**, donde cada hormiga ejecuta tareas específicas bajo una estrategia central.

## Correspondencia Metáfora ↔ Sistema

| Elemento de la Metáfora | Representación en AntCiberDron |
|-------------------------|--------------------------------|
| Hormiguero | Sistema completo gestionado por la KGD |
| Hormiga | AntCiberDron especializado |
| Memoria del hormiguero | Archivo `Grupo02.csv` con coordenadas y arsenal |
| Cerebro de la hormiga | Autómata Finito Determinista (BBA) |
| Ataque coordinado | Evaluación del autómata y explosión de la bomba |
| Señales internas | Alertas y resultados True / False |
| Exploración | Lectura secuencial de coordenadas con loading |

Esta metáfora permite al equipo entender el sistema como una **estructura cooperativa, automática y orientada a decisiones**, alineada con la filosofía XP.
